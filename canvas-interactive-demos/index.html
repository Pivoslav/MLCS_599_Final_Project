Mechanic brainstorm demos

Each page is playable vanilla HTML; paste into Canvas via the parent canvas-rce-embed-fragment pattern or link out to GitHub Pages.

  • Winter / crisis lab

    Rumor threshold preview + collapsible die table (path × d6).

    Fun: puzzle beat before the fiction moves. Educational: probability and institutions in one view—why “we needed Reform higher” is also a historical claim.

  • Path deltas + pedagogy hint

    Same choice label, different Order/Reform/People effects per path; optional gloss under details.

    Fun: the verb stays, the numbers dance. Educational: fights the single “right move” for 1836; surfaces Slavophile vs Westernizer costs.

  • Breadcrumb + tension meter

    Scene strip + “spread” bar (max − min across the three realms).

    Fun: quick situational read. Educational: co-op debrief hook—“who’s winning the argument numerically?” without opening modals.

  • Co-op timer, ballot, stance gate

    30s cue, hidden votes then reveal, checkbox before “commit.”

    Fun: committee drama. Educational: models salon equity; reduces one player owning every click.

  • Debrief: thesis + compare + branch peek

    Side-by-side epilogue stubs, pick a historian one-liner, read-only other-path snippet in dialog.

    Fun: re-title your run. Educational: contingency and lens; FOMO relief without save-scumming.

  • Achievement toast + token icon flash

    Rare beat fires status toast and drops an SVG token with a short motion accent.

    Fun: collectible moments. Educational: mnemonics back to flood / file / bridge motifs in Pushkin–Chaadaev space.

  • URL scene hint + copy summary

    ?scene= banner + clipboard-friendly run blurb.

    Fun: shareable. Educational: instructor deep links; forum posts that force path + meters + ending in one line.
