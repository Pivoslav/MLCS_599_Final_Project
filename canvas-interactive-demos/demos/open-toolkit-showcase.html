Open toolkit showcase

This page intentionally loads a few libraries from public CDNs (Mermaid, Three.js, Google model-viewer). Other demos in this folder stay CDN-free for picky LMS hosts; use this lab on GitHub Pages or when your course allows external script.

Ideas, not course requirements: STL/3D, diagrams, and archive embeds are spice—pair with SOURCES.md for rights. For 3D prints of real monuments, prefer models you have permission to redistribute or CC-licensed meshes.

2. Embedded STL — 3D in the syllabus (and at the makerspace)

ASCII STL is just text; you can version it in GitHub, diff it, and parse it in the browser with Three.js. Same mesh can feed a slicer for a tactile “Bronze Horseman” / Neva flood discussion object—STL is not only for engineering courses.

3. model-viewer — GLB/GLTF (lighter than raw STL for the web)

Google’s <model-viewer> handles AR on some devices. For this course lab, the intended mesh is a table lantern (salon candlelight), not a street lamp: MakerWorld — medieval table lantern (LED candle). MakerWorld/Bambu models ship as print files; export to GLB yourself (e.g. Blender: import STL/3MF → export glTF binary), then drop it next to this demo as canvas-interactive-demos/assets/medieval-table-lantern.glb. This page auto-loads that file when it exists; otherwise it falls back to a generic Khronos lantern sample. modelviewer.dev

Preferred mesh: MakerWorld #1066715 — follow the license / terms on that page before redistributing; this repo does not bundle their geometry. Local path: ../assets/medieval-table-lantern.glb (from this HTML file). Fallback mesh: Khronos Lantern (sample street-style lantern; CC BY 4.0 — confirm in folder README). 4. Web Audio — one-shot ambient without MP3 files

Oscillators, short noise bursts, and envelopes only—no MP3. Useful for tiny UI stingers in the CYOA (winter beat, library token). All play on button click (user gesture unlocks audio). Do not auto-play beds; pair any future loop with prefers-reduced-motion and a visible mute.

Pedagogy: these are evocative, not documentary—discuss sound as framing, not as period recording. For real period music or speech, use licensed files and cite in SOURCES.md.
