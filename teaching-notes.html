Teaching & LMS integration

Students: use the main game at index.html (or your GitHub Pages root). This page is for instructors and TAs.

Canvas

  1. Open or create a Page in your course.
  2. Edit → switch to the HTML / raw editor (e.g. </> or “HTML Editor”).
  3. Copy the whole file from canvas-rce-embed-fragment.html and paste into the editor.
  4. Save / Publish, then check Student View: “Open game in new tab” opens the full GitHub Pages layout.
  • The fragment is link-first (full browser window for realm meters and three-player co-op). Optional iframe: see HOSTING-AND-INTEGRATION.md.
  • Full checklist and sizing notes: HOSTING-AND-INTEGRATION.md §1.
  • Before the term, optionally embed canvas-interactive-demos/demos/canvas-self-test.html once in a draft page to see LMS quirks.

Directed start (?scene=)

Append ?scene=<sceneId> to the game URL. IDs are keys on the scenes object in js/game-scenes.js. Skips earlier beats, use for labs, not first play. Details: README.md.
